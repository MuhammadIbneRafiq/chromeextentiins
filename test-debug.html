🤖 AI Guardian Debug Test Page

🔍 How to See Debug Information

1. Browser Console (F12): Press F12 or right-click → Inspect → Console to see all debug logs

2. Extension Popup: Click the extension icon and check the "🐛 Debug Console" section

3. Service Worker Console: Go to chrome://extensions → Find "AI Productivity Guardian" → Click "service worker" link

💡 Pro Tip

Keep the browser console open (F12) while navigating to see real-time debug information from the AI Guardian extension!

📊 What You Should See

In Browser Console: Messages starting with "[AI Guardian Debug]" and styled logs

In Extension Popup: Recent debug logs in the Debug Console section

In Service Worker: Detailed background processing logs
